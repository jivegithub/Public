# 新生ポイントプログラム エントリー（16桁のT会員番号のみ対応）
新生ポイントプログラム エントリーの自動化スクリプト
* shinsei_point_program
    * shinsei_point_program.rb
        * エントリー時に実行するファイル
        * 事前にファイル内の入力情報（口座番号やT会員番号）の部分の編集が必要
    * lib/entry.rb
        * エントリー処理を行うスクリプト（変更不要）
## 環境
* Windows
* Ruby
* Selenium
* GoogleChrome
